## 소개

클로드 코드로 생산성 증가, 워크플로우 자동화를 위한 요소와 문제 해결법을 담은 공간입니다.
프로젝트에서 사용하고 있는 에이전트/커맨드/스킬/템플릿을 계속 업데이트합니다.
필요한 컨텍스트로 제공하는 코드도 업로드할 예정입니다.

## 요소별 안내

### 🪄 Skills

특정 워크플로우를 자동화하기 위해 만든 스킬입니다.

### 🤖 Agents

특정 반복적 & 병렬 가능 업무에 특화된 에이전트입니다.

### 🚄 Commands

자주 사용하는 프롬프트를 작은 단위로 분리한 커맨드입니다.

### 📚 Docs

가이드를 관리합니다.

- ex)
  - `/setup.md`: 설치 및 설정 가이드
  - `/best-practices/**`: 발견한 베스트 프랙티스
  - `/trouble-shooting/**`: 문제 해결 가이드

### 🪝 Hooks

특정 이벤트 발생 시 자동으로 실행되는 훅 스크립트입니다.

### 📊 Metrics

성과 추적을 위한 측정 지표입니다.
요소들의 이름과 똑같은 `.md` 문서에 각 파일 단위로 지표를 기록합니다.

**🔄 공통 지표 - 생산성**

- 프롬프팅 카운트: 목표 태스크 하나당 해당 요소로 시작하는 프롬프트 사용 횟수
- 재사용률: 스킬/에이전트/커맨드의 재사용 횟수

**🔬 개별 지표 - 도메인별 상이함**

- 버그 조기 탐지율(%): 커밋 전 자동 검증으로 발견한 이슈 비율
- 테스트 커버리지(%): 자동 생성된 테스트로 증가한 커버리지
- 코드 컨벤션 일치율(%): 서버 액션, 컴포넌트, zod 스키마 등 각 요소간 코드 패턴 일치율

### 📝 Templates

프로젝트에서 사용하는 코드 템플릿과 가볍지 않은 프롬프트입니다.
컨텍스트 이해를 위한 목적으로 업로드합니다.
